# Regular Expression to DFA
Turning a regular expression to a NFA and then to a DFA

To make 

All: 
$ make

regex:
$ make regex_main

ls: 
$ make ls_main

To Use

regex:
$ ./regex_main

ls:
$ ./ls_main

For help

regex:
$ ./regex_main -h

ls:
$ ls_main -h
